# FingeRun
Example of a Mobile Based Type of Game

# Description
This is a Video Game Project which aims to simulate a car ride from the perspective of the passanger while using his imagination by looking at the window. Use the touchscreen controls to avoid cars, flies and other obstacles before reaching the end of the level.

# Features
- Levels of pure skill.
- A walking hand.
- Touchscreen controls.
- A lot of animations.

# Programs Used
- Unity 6
- C# (Visual Studio)

I've learned how to make a mobile game with touchscreen controls.
